2012/11/11

この度は本ソフトをダウンロードして頂きありがとうございます。

・・・というのは建前で、需要が少ないのにわざわざ一晩費やしてGUI化したので
結構投げやりです(:3_ヽ)_

■このソフトは？
コマンドラインと疎遠な人でもログのhtmlのソースから
コダマの選出数・勝利数・勝率・ユーザー数を割り出します。
特定ユーザーに絞った出力も可能。


■動作環境
Java Runtime Environment Version 7 以上
Javaさえ入っていればOSを問わず動作しますが、
JRE6以下は恐らく動作しません。
動作しない場合はJRE6をアンインストールし、
http://java.com/ から再インストールしてください。


■使い方
1.玉神楽の見たいログを開く

2.右クリック→「ソースを表示」からソースを表示する(ブラウザによって違うかも？)

3.Ctrl+A等でとにかくたくさんコピーする
※誤読防止のため、「<table」の文字を見つけるまではスルーするので
  よく分からない場合は全部コピペ推奨

4.テキストエディタを開いてペーストして保存(仮にdata.txtとします)

5.本プログラムを起動

5.5.特定IDのデータだけ見たい場合はIDを入力

6.data.txtをプログラムのどこでもいいのでドラッグアンドドロップ

7.それっぽいcsvファイルが出来上がるので
  エクセルなりOpenOfficeなりテキストエディタで読みましょう

8.文字化けする場合はエンコーディングを変えましょう
